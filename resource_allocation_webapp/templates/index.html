Cloud Resource Allocation Simulation

This interactive simulation demonstrates an intelligent cloud resource allocation framework based on machine learning concepts. The system is designed to adjust CPU resource allocation dynamically based on simulated workload data.

Simulation Parameters

Please specify the number of simulation steps you would like to run. Increasing the steps provides a more detailed simulation, but may increase computation time.



About the Simulation

Background: Based on research on resource allocation optimization in cloud computing environments, this simulation uses a dummy agent (a heuristic-based predictor) to mimic dynamic allocation. The technical goal is to minimize the penalty between allocated CPU resources and actual demand, while considering operational costs.

Inputs: The primary input is the number of simulation steps. Future versions could allow more detailed configuration, such as adjusting penalty weights and demand profiles.
